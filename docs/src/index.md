# brother-ql-node

Welcome. This site helps you ship printing in your app with `brother-ql-node`, then go deeper only when you need protocol details.

`brother-ql-node` is a TypeScript workspace for Brother QL printers on modern Node.js, with parity goals inspired by the original Python project [`pklaus/brother_ql`](https://github.com/pklaus/brother_ql).

## Start here

- New to the library: read [App integration](./guide/app-integration.md)
- Using the CLI surface: read [CLI usage](./cli/overview.md)
- Running into issues: read [Troubleshooting](./guide/troubleshooting.md)

## What this docs site covers

- Quick setup for TCP and USB printer flows
- Current API behavior in `@brother-ql/node`
- Current command surface in `@brother-ql/cli`
- Practical constraints so you can avoid common integration mistakes

## Reference notes

Implementation homework and parity analysis live in `docs/analysis/`. They are useful for maintainers and deep debugging, but not required for first-time setup.
